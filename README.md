## Getting Started
Follow the steps below to set up the environment and run the application locally.

### 1. Installation
Ensure you have [Node.js](https://nodejs.org/) installed, then execute:
```
npm install
```

### 2. Running
```
npm start
```

### 3.Testing
```
http://localhost:8099/
```
